Declares a property.

The first parameter is the name of the property. Capitalization is recommended. The second parameter is the type of the property. The macro will define according getter and setter methods.

For example: 

  Property(Name, std::string)

will generate 

   std::string getName() const { ... }
   void setName(std::string value) { ... }
