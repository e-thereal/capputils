Detailed Description

template<class T, bool fromMethod = true>
class capputils::reflection::Converter< T, fromMethod >

Basic template for a Converter.

Converters are used to convert the value of a property to a string and vice versa. The fromString() method need not to exists. This is handled by the template value parameter fromMethod which defaults to true.
